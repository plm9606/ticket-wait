# 크롤러 실행

공연 정보 크롤러를 실행합니다. 대상 플랫폼을 선택하거나 전체를 실행할 수 있습니다.

사용법: `/crawl [all|interpark|yes24|melon]` (기본값: all)

## 실행 방법

1. `apps/server` 디렉토리에서 `pnpm exec tsx src/crawlers/test.ts $ARGUMENTS` 명령을 실행합니다.
2. 실행 결과(발견 건수, 신규 건수, 매칭 결과, 에러)를 사용자에게 보고합니다.
3. 에러가 발생한 경우 원인을 분석하여 설명합니다.

## 주의사항

- DB(PostgreSQL)가 실행 중이어야 합니다. 연결 실패 시 `docker compose up -d`를 안내합니다.
- 크롤링은 rate limiting(요청 간 3초 간격)이 적용되어 시간이 걸릴 수 있습니다. 특히 멜론은 상세 페이지까지 크롤링하므로 가장 오래 걸립니다.
- 크롤링 후 자동으로 아티스트 매칭이 수행됩니다.
